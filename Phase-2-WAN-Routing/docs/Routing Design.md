The campus network continues to use DSW1 as the Layer 3 gateway for VLANs 10, 20, and 99.

RTR-2 functions as the campus edge router connecting the LAN to the WAN.

RTR-3 provides WAN transit between headquarters and the remote branch.

RTR-4 represents the remote branch router.

Static routing is implemented first to reinforce routing fundamentals before migrating the network to OSPF.
