Topology Summary

- DSW1 connects the campus LAN to RTR-2 through VLAN 99.
- RTR-2 serves as the headquarters edge router.
- RTR-3 acts as the WAN transit router.
- RTR-4 represents a remote branch router.
- Routers communicate over serial point-to-point links.
